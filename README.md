# finhabit_front
2학기 프로젝트 Finhabit 프론트엔드 레포

git bash에 아래 명령어 입력하셔서 시작하시면 됩니다!<br>
npm run dev

# Commit Message Convention
<ul>
  <li>Init : 프로젝트 세팅 <br> ex) Init - issue template 수정</li>
  <li>Feat : 새로운 기능 추가 <br> ex) Feat - 로그인 페이지 구현</li>
  <li>Refactor : 코드 리팩토링 <br> ex) Refactor - 회원가입 코드구조 개선</li>
</ul>

# Branch Name Convention
이유 : PR시 한글이 포함된 브랜치명의 경우 경고를 띄움 / 이동의 원활을 위해<br>
[이슈번호]-[작업간단설명]<br>
ex)17-quizpage_modification
